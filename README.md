\# Quiz App MVP



Веб-приложение для создания и проведения квизов в реальном времени.



\## Описание проекта



Проект представляет собой MVP веб-приложения для проведения интерактивных квизов. В системе есть две роли пользователей: организатор и участник.



Организатор может создавать квизы, добавлять вопросы, настраивать время ответа, запускать live-комнату и смотреть результаты участников. Участник может подключиться к активному квизу по коду комнаты, отвечать на вопросы в реальном времени и видеть результат.



\## Основные функции



\- регистрация и авторизация пользователей;

\- роли пользователей: организатор и участник;

\- личный кабинет для организатора и участника;

\- создание, просмотр и удаление квизов;

\- добавление вопросов с одним правильным ответом;

\- добавление вопросов с несколькими правильными ответами;

\- добавление изображений к вопросам;

\- настройка категории квиза;

\- настройка времени ответа на вопрос;

\- запуск live-комнаты;

\- подключение участников по коду комнаты;

\- показ вопросов в реальном времени;

\- таймер ответа;

\- отправка ответов участниками;

\- автоматическая проверка ответов;

\- подсчёт баллов;

\- отображение лидерборда;

\- история участия и проведённых квизов.
